# Security Policy

## Supported Material

This repository contains public examples, benchmark scripts, and integration notes for AeroPivot. It does not contain the proprietary AeroPivot SDK source code.

For AeroPivot SDK security questions, license questions, or private disclosures, contact Kasmav through:

- Website: https://kasmav.com/aeropivot
- Support: https://kasmav.com/aeropivot/docs

## Reporting a Vulnerability

Please do not open a public issue for sensitive security reports. Send a private report with:

- affected package, example, or documentation page
- steps to reproduce
- browser and operating system
- screenshots or logs when useful
- expected impact

We will acknowledge valid reports and coordinate a fix or documentation update.

## Example Data

Do not commit private datasets, customer files, license keys, production API endpoints, or personally identifiable information when adapting these examples.
