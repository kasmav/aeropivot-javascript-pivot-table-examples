# Contributing

Contributions are welcome for public examples, benchmark scripts, documentation improvements, and framework integration notes.

## Guidelines

- Do not commit AeroPivot commercial SDK binaries.
- Do not commit license keys.
- Do not commit private datasets or customer data.
- Keep examples small enough for developers to understand quickly.
- Include browser, operating system, hardware, row count, field count, and field cardinality when discussing benchmark results.

## Suggested Contributions

- additional React, Vue, Angular, Svelte, or vanilla JavaScript examples
- CSV, JSON, Arrow, or Parquet ingestion patterns
- benchmark environment templates
- documentation fixes
- integration notes for backend-rendered apps
