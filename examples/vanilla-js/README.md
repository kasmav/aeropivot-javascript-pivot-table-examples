# Vanilla JavaScript Pivot Table Example

Plain HTML and JavaScript integration for the AeroPivot `<pivot-table>` web component.

This is the best starting point for static sites, server-rendered applications, PHP, Django, Ruby, Laravel, Rails, ASP.NET Razor, and other stacks where the browser can load standard JavaScript modules.

## Setup

Copy AeroPivot SDK files into:

```txt
aeropivot/pivot-table.es.js
aeropivot/pivot-themes.css
aeropivot/assets/pivot.worker.js
```

Then open `index.html` in a browser or serve the folder with any static server.
