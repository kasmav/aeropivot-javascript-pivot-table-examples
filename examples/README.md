# AeroPivot Examples

These examples show how to embed the AeroPivot `<pivot-table>` web component in common JavaScript application stacks.

## Examples

| Example | Status | Notes |
| --- | --- | --- |
| `react` | Runnable Vite example | Best starting point for React and Next.js developers. |
| `vanilla-js` | Static browser example | Best starting point for plain HTML, server-rendered apps, PHP, Django, Ruby, and static sites. |
| `vue` | Integration notes and SFC pattern | Shows Vue 3 refs, events, and setup timing. |
| `angular` | Integration notes and component pattern | Shows `CUSTOM_ELEMENTS_SCHEMA`, `ViewChild`, and setup timing. |

## SDK Files

The examples expect you to download the AeroPivot trial or use your licensed SDK files:

- `pivot-table.es.js`
- `pivot-themes.css`
- `assets/pivot.worker.js`

Download: https://kasmav.com/aeropivot/download

Do not commit those SDK files to this public repo.
