# Support

This repository is a public companion kit for AeroPivot examples and benchmark methodology.

## Where To Get Help

- Product: https://kasmav.com/aeropivot
- Documentation: https://kasmav.com/aeropivot/docs
- Integration guides: https://kasmav.com/aeropivot/docs/guides
- Trial download: https://kasmav.com/aeropivot/download
- Benchmark article: https://kasmav.com/aeropivot/benchmarks/10-million-row-pivot-table

## Good GitHub Issues

Open an issue for:

- unclear example setup steps
- benchmark script problems
- missing framework integration notes
- documentation improvements

For commercial license support, private data issues, or production debugging, use the support path provided with your AeroPivot license or trial.
